# `fork()` 시스템 호출 정리

## 📌 개요

`fork()`는 현재 실행 중인 프로세스를 **자기복제(自己複製, self-replication)** 하여 **자식 프로세스를 생성**하는 시스템 호출입니다.

---

## ⚙️ 작동 방식

- `fork()`는 **한 번 호출되지만, 두 번 반환**됩니다.

### 반환 값

| 실행 주체       | 반환 값                           |
|----------------|-----------------------------------|
| 자식 프로세스   | `0`                                |
| 부모 프로세스   | 자식 프로세스의 PID (양의 정수)     |
| 실패한 경우     | `-1` (자식 생성 실패, errno 설정됨)  |
